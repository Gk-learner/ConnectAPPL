# 🚀 Connect App

A full-stack MERN application designed to connect users through a modern, responsive interface. This project demonstrates end-to-end development skills, including frontend UI, backend APIs, and database integration.

---

## 🌟 Features

- 🔐 User Authentication (Login/Signup)
- 👤 User Profile Management
- 🔄 Real-time / dynamic data handling
- 🌐 RESTful API integration
- 📱 Responsive and modern UI
- ⚡ Fast and scalable architecture

---

## 🛠️ Tech Stack

**Frontend:**
- React.js
- JavaScript (ES6+)
- CSS / Tailwind 

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB

**Other Tools:**
- Cursor (AI-assisted UI development)
- Git & GitHub

---
## ⚙️ Setup

### Backend
cd connect-appBE  
npm install  
npm run dev  

### Frontend
cd connect-appFE  
npm install  
npm run dev
